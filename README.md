# StoryMaker

Dispositivo educativo basado en Raspberry Pi Zero 2W para talleres literarios.
Genera premisas narrativas aleatorias y las muestra por pantalla e-ink, impresora térmica o audio.

## Estado
En desarrollo activo. Este repo es el espacio de trabajo privado.

## Estructura
- `firmware/` — Código Python para la Raspberry Pi
- `hardware/` — STL, esquemas eléctricos, lista de componentes
- `docs/` — Manual, guías de montaje, imágenes
- `releases/` — Notas de versión
